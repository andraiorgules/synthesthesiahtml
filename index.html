Synthesthesia

A JUCE Framework Project

Introduction

Synthesthesia is a simple Synth plugin for people who are beginning their journey into the world of audio.
It consists of:
- 1 oscillator that can switch between a Sine, Sawtooth, and Square wave
- An envelope that modulates the oscillator signal
- A filter component that can switch between low-pass band-pass, and high-pass filters
- An envelope that modulates the filter

It also features:
- A waveform that changes along with the parameters chosen by the user
- Pop-up buttons that explain what each component is and what it does.
Synthesthesia is available as a VST3, AU, and Standalone plugin.

Methods and processes applied

There was a lot of trial and error involved in this project. I did a lot of reasearch and tried to implement different components that ended up not being compatible with my software version, or just not working for some reason. I definitely had to do some digging to find ways to implement the features that I wanted. Ultimately, I ended up with 3 component classes (ADSR, Filter, OSC) for the UI, and 3 for the backend Data. My Data classes connect to my SynthVoice class (which is responsible for the actual sound), and my UI classes connect to my Plugin Editor (which is responsible for the UI). The Plugin Processor class is responsible for tracking changes made by the user and updating the parameters accordingly.

Findings / concluding thoughts

I was sad to learn that some resources online are outdated and were incompatible with my software version. However, I think this is an interesting project and I would like to expand on it more in the future.
